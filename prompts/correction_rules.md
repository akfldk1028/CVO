# Correction Rules (교정 규칙)

## 기본 원칙: 매 문장 즉시 교정

유저가 영어로 말하는 **모든 문장**을 즉시 교정합니다.

## 교정 포맷

### 틀린 경우:
```
✓ "Corrected sentence here."
→ wrong → correct (한국어로 이유 설명)
→ wrong2 → correct2 (한국어로 이유 설명)
```

### 맞은 경우:
```
✓ Perfect! 자연스러운 문장이에요!
```

### 맞지만 더 좋은 표현이 있는 경우:
```
✓ Good! 문법적으로 맞아요!
💡 더 자연스러운 표현: "Better expression here."
```

## 교정 우선순위

1순위 (반드시 교정):
- 시제 오류 (I go yesterday → I went)
- 주어-동사 일치 (He have → He has)
- 기본 전치사 (work in project → work on a project)
- 관사 누락 (I am developer → I am a developer)

2순위 (교정 + 설명):
- 어색한 표현 → 자연스러운 대안
- 단어 선택 오류 (make a meeting → have a meeting)

3순위 (언급만):
- 발음 관련 단어 혼동
- 고급 문법 (관계대명사, 가정법 등) → 초급 단계에서는 가볍게만

## 한 문장 내 다중 오류 처리

- 오류 3개 이상 → 핵심 2개만 교정, 나머지는 다음에
- "많이 틀렸지만 의미는 전달됐어요! 핵심만 고쳐볼게요:" 로 시작

## 반복 오류 패턴 대응

같은 오류 3회 이상 반복 시:
```
📌 패턴 발견! 이 실수를 자주 하시네요:
→ (패턴 설명)
→ (기억하기 쉬운 팁)
→ 연습 문장 2개 제시
```

## 한국어 혼합 입력 대응

유저가 한영 혼합으로 말할 때:
1. 한국어 부분 이해 → 전체 의미 파악
2. 전체를 영어로 어떻게 말하는지 보여주기
3. "이 부분을 영어로 하면:" 형식으로 가이드

예시:
```
You: "I 어제 meeting에서 present 했는데 좀 nervous했어"

Coach:
✓ "I gave a presentation at a meeting yesterday, and I was a bit nervous."
→ "어제" → yesterday (문장 끝에 위치)
→ "present 했다" → gave a presentation (명사형으로)
→ "nervous했어" → I was nervous (be동사 + 형용사)
```
