# Digital Twin Technical Terms (디지털트윈 기술 용어)

## 핵심 디지털트윈 용어
| 영어 | 한국어 | 면접 예문 |
|------|--------|----------|
| Digital Twin | 디지털트윈 | "A digital twin is a virtual replica of a physical asset." |
| Physical asset | 물리적 자산 | "We create digital twins of physical assets like buildings." |
| Real-time synchronization | 실시간 동기화 | "The system supports real-time synchronization with sensors." |
| Simulation | 시뮬레이션 | "We run simulations to predict equipment failures." |
| Predictive maintenance | 예측 정비 | "Digital twins enable predictive maintenance." |
| Data visualization | 데이터 시각화 | "I built a 3D data visualization dashboard." |

## 3D / 렌더링
| 영어 | 한국어 | 면접 예문 |
|------|--------|----------|
| 3D rendering | 3D 렌더링 | "I optimized 3D rendering for real-time performance." |
| Level of Detail (LOD) | 세부 수준 | "We implemented LOD to improve frame rates." |
| Mesh optimization | 메시 최적화 | "I reduced polygon count through mesh optimization." |
| Texture mapping | 텍스처 매핑 | "We used PBR texture mapping for realistic materials." |
| Scene graph | 씬 그래프 | "The scene graph manages all 3D objects hierarchically." |
| Frame rate / FPS | 프레임 레이트 | "We maintained 60 FPS on the target hardware." |
